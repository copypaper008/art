## Recognition Engine

An interactive installation that face-swaps gallery visitors onto historical
queer figures and issues a mock bureaucratic verdict.

→ See [`recognition-engine/README.md`](recognition-engine/README.md) for full setup, running, and development instructions.

**Quick start:**
- macOS: double-click `recognition-engine/start-mac.command`
- Windows: double-click `recognition-engine/start-windows.bat`
- Manual: `cd recognition-engine && python3 server.py & python3 -m http.server 8080 &` then open `http://localhost:8080/?stations=4`
